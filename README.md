# Umrah Supermarket Web

Frontend application for the Umrah Supermarket website.

Built with React and Vite, this app includes the main marketing and shopping experience: home page, category browsing, grocery listing, about, and contact pages.

## Tech Stack

- React
- React Router
- Tailwind CSS
- Vite
- Spline (for selected 3D visuals)

## Run Locally

1. Install dependencies:

```bash
npm install
```

2. Start development server:

```bash
npm run dev
```

3. Build for production:

```bash
npm run build
```

4. Preview production build locally:

```bash
npm run preview
```
